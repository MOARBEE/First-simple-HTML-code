# First-simple-HTML-code
This is the most basic start to HTML, which consists of a basic title and a heading and a paragraph.
<!DOCTYPE html> defines the document type and HTML version. This is just a standard
<html> This encloses the full HTML document and mainly comprises of document header which is represented <head>...</head> and document body which is represented by <body>...</body> tags.
<head> This tag represents the document's header which can keep other HTML tags like <title>, <link> etc
<title> This is used inside the head and and mentions the title of the document
<body> This tag represents the document's body which keeps other HTML tags like <h1>, <div>, <p> etc
<h1> This tag represents a heading, you also get <h2>, <h3> etc but the headings keep decreasing in size, so h1 is big, h2 is smaller and so on
<p> This just means a paragraph
